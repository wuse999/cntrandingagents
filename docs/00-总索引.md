# cntrandingagents 文档总索引

## 1. 文档用途

- 作为 `cntrandingagents` 全套治理文档的统一入口。
- 固定本项目当前唯一目标是“对官方 `TradingAgents` 做中文汉化”，不承担功能增强。
- 固定本项目当前的协作模式、仓库角色、同步路径、发布口径、凭据基线与 AI 执行基线。
- 最后更新时间：2026-04-21

## 2. 项目一句话定位

`cntrandingagents` 是基于官方 `TauricResearch/TradingAgents` 的中文汉化专用仓，目标是尽量不改变行为，只做文档、CLI、帮助文本与对外展示文案的中文化，并保持后续可直接同步官方上游。

## 3. 当前固定协作模式

- Windows：本地镜像 / 应急副本 / 文档入口，不是默认主工作树。
- Ubuntu：唯一正式主工作树。
- GitHub：主仓与主发布仓。
- Gitee：镜像仓与国内发布源。
- `raw`：官方上游镜像，只用于对照与同步，不承载汉化主线。
- `cntrandingagents`：你的正式汉化项目仓。
