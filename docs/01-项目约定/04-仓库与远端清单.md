## 5. 默认分支与命名约定

- 默认主分支：`main`
- 上游同步分支建议：`sync/upstream-YYYYMMDD` 或 `sync/<official-tag>`
- 汉化整理分支建议：`l10n/<topic>`
- 发布 tag 建议：`vX.Y.Z-cntrandingagents.N`

## 6. 默认 Git 角色说明

- `upstream`：官方同步源
- `origin`：GitHub 主仓
- `gitee`：Gitee 镜像仓

## 7. 首次检查命令

在 Ubuntu 主工作树执行：

```bash
pwd
git branch --show-current
git remote -v
git status --short --branch
git ls-remote origin refs/heads/main
git ls-remote gitee refs/heads/main
git ls-remote upstream refs/heads/main
```

在 Windows 本地镜像执行：

```powershell
git -C D:\tranding\tranding_agents_cn\cntrandingagents status --short --branch
git -C D:\tranding\tranding_agents_cn\cntrandingagents remote -v
```
